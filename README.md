# Image Captioning with Flickr8k

## 📌 Introduction
This project builds an image captioning model using CNN + LSTM on the Flickr8k dataset.

## 🚀 Features
- Extract image features using VGG16
- Generate captions using LSTM
- Train from scratch on Flickr8k

## 🛠️ Technologies
- Python
- TensorFlow / Keras
- Google Colab

## 📊 Dataset
- Flickr8k dataset

## ▶️ How to run
1. Open KLTN.ipynb in Google Colab
2. Upload dataset
3. Run all cells
